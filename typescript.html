Typescript example

In this example, you'll see a drop target below, and next to that an image display area.

Drop an image onto that drop target and it'll be loaded into the image view area straight in the page without needing to use the server.

Once you have an image you like, click the upload button to actually send it to the server as a post upload.

If you press F12 to open your debugger console, you'll see info messages telling you which events are occuring.
